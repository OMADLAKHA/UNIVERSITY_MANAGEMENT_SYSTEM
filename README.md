# UNIVERSITY_MANAGEMENT_SYSTEM

Project Overview:
The Automatic Timetable Management System is a console-based application developed using the C programming language. This project automates the creation of class schedules and the management of faculty and student records for a school.

It allow to:

1:- Add and maintain student records
2:- Manage and add faculty records
3:- Assign subjects to faculty
4:- Automatically generate a timetable
5:- Look up and see student schedules
6:- Look up and see faculty schedules
7:- The project uses file handling for permanent storage of data and follows a modular programming structure using multiple .c and .h files.

Features: 

1:- Menu-driven console interface
2:- Automatic batch assignment and student ID
3:- Faculty subject allocation
4:- Randomized, conflict-free schedule creation
5:- Student timetable search by name
6:- Faculty timetable search by name
7:- Data stored permanently using binary files
8:- Header files and a modular code structure
